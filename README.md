# expenseTracker
Created an expense tracker in Python

Description:
A command-line expense tracker that helps users efficiently record and monitor their personal spending. The project demonstrates practical skills in Python programming, file handling, and data manipulation.

Key Features:

1: Add Expense:

   Record amount 
   
   category
   
   description
   
   date (user input or automatic)
   
2: View Expenses:

  Filter expenses by date.
  
  Filter expenses by category.
  
  View the last X entries.
  
  Calculate and display total spending.
  
3: Data Persistence:

  Stores all expenses in a JSON file.
  
  Ensures data is saved between program runs.
